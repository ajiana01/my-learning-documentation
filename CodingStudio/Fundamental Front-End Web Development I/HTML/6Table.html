Penjelasan dari tag-table dan atribut-atributnya adalah sebagai berikut:
tag pembuka untuk tabel : tag untuk membuat baris pada tabel : tag untuk membuat header pada tabel. Biasanya digunakan untuk menampilkan nama kolom. : tag untuk membuat sel pada tabel. Digunakan untuk menampilkan data pada tabel. Atribut-atribut yang sering digunakan pada tag-table adalah: border: menentukan tebal garis batas tabel. Contoh: border="1" cellspacing: menentukan jarak antara sel-sel pada tabel. Contoh: cellspacing="0" cellpadding: menentukan jarak antara batas sel dan isi sel pada tabel. Contoh: cellpadding="5" width: menentukan lebar tabel. Contoh: width="100%" Selain itu, kita juga dapat menggabungkan beberapa sel pada tabel dengan menggunakan atribut rowspan dan colspan. Atribut rowspan digunakan untuk menggabungkan beberapa baris pada tabel, sedangkan atribut colspan digunakan untuk menggabungkan beberapa kolom pada tabel. Contoh penggunaan atribut rowspan dan colspan:
Nama Alamat
Jalan Kota
Alice Jl. Kenangan Surabaya
Bob Malang
