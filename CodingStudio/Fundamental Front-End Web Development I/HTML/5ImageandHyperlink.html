Tag Image (): Tag ini digunakan untuk menampilkan gambar pada halaman web. Berikut adalah sintaks dasar dari tag image: Penjelasan dari atribut-atribut pada tag image adalah sebagai berikut: src: atribut ini digunakan untuk menentukan URL atau lokasi file gambar yang akan ditampilkan pada halaman web. alt: atribut ini digunakan untuk memberikan deskripsi alternatif dari gambar, yang akan ditampilkan jika gambar tidak dapat ditampilkan atau jika pengguna menggunakan layanan pembaca layar. width dan height: atribut-atribut ini digunakan untuk menentukan ukuran gambar yang akan ditampilkan pada halaman web. Tag Hyperlink (): Tag ini digunakan untuk membuat tautan atau hyperlink ke halaman web atau dokumen lain. Berikut adalah sintaks dasar dari tag hyperlink: teks_tautan Penjelasan dari atribut-atribut pada tag hyperlink adalah sebagai berikut: href: atribut ini digunakan untuk menentukan URL atau lokasi halaman web atau dokumen lain yang akan dihubungkan dengan tautan. teks_tautan: teks yang ditampilkan pada tautan.
