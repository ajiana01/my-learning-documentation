Tag input HTML digunakan untuk membuat input form pada halaman web, seperti textbox, checkbox, radio button, dan sebagainya. Berikut adalah contoh sintaks dari tag input:
Remember Me Male Female
Penjelasan dari beberapa atribut yang sering digunakan pada tag input adalah sebagai berikut:
type: menentukan jenis input form yang akan ditampilkan. Beberapa jenis input form yang sering digunakan antara lain: text, password, checkbox, radio, submit, dan reset.
name: menentukan nama variabel yang akan digunakan untuk mengambil data dari input form.
value: menentukan nilai atau isi dari input form.
placeholder: menampilkan teks bantuan atau petunjuk di dalam input form, contohnya dalam kasus pengisian username atau password.
Selain itu, terdapat juga beberapa atribut lain yang dapat digunakan pada tag input, seperti 'disabled' untuk menonaktifkan input form, 'required' untuk membuat input form wajib diisi, dan 'readonly' untuk membuat input form tidak bisa diubah nilainya.
