1. Selector elemen: Selector ini mengacu pada elemen HTML tertentu, seperti h1, p, div, dan sebagainya. Contoh penggunaannya adalah:

    h1 { font-size: 24px; color: blue; }

  2. Selector class: Selector ini mengacu pada elemen HTML yang memiliki class tertentu. Contoh penggunaannya adalah:

    .class-name { font-weight: bold; text-align: center; }

  3. Selector ID: Selector ini mengacu pada elemen HTML yang memiliki ID tertentu. ID harus unik di seluruh halaman HTML. Contoh penggunaannya adalah:

    #id-name { background-color: yellow; padding: 10px; }

  4. Selector atribut: Selector ini mengacu pada elemen HTML yang memiliki atribut tertentu. Contoh penggunaannya adalah:

    input[type="text"] { border: 1px solid black; padding: 5px; }

Pembacaan pada css secara terurut apabila pernah menggunakan style kedua kalinya makan style paling bawah yang digunakan!
