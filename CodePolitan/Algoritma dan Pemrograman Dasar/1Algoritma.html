Algoritma dan Pemrograman Dasar

Karakteristik algoritma diantaranya:

  1. Terbatas(Finiteness)

    2. Proses dalam algortima harus memiliki akhir dan jumlah Terbatas

  2. Tidak Ambigu (Definiteness)

    3. Setiap langkah proses dalam algortima harus jelas dan tidak ambigu

  3. Input dan Output

    4. algortima harus menerimal nol atau lebih input dan setidaknya menghasilkan satu output

  4. Efektif (Effectiveness)

    5. langkah - langkah dalam algoritma harus efektif dan sederhana mungkin

Bahasa Pemrograman

Sebuah intruksi standar untuk memerintah komputer agar bisa menjalankan fungsi tertentu.

Fungsi Bahasa Pemrograman

untuk memerintah komputer agar bisa mengolah data sesuai dengan yang kita inginkan

Generasi bahsa Pemrograman

  1. Generasi Pertama (Machine language)
  2. Generasi Kedua (Assembly language)
  3. Generasi ketiga (Procedural language)
  4. Generasi kerempat (4GL) Fouth-generation language
  5. Generasi kelima (Programming language based object oriented & web development)

Tingkatan bahasa Pemrograman

  1. Bahasa tingkat rendah. Contoh: Bahasa Mesin.
  2. Bahasa tingkat menengah. Contoh: Bahasa Assembly
  3. Bahasa tingkat tinggi. Contoh: C++, VB, Delphi, Pascal
  4. Bahasa tingkat object. Contoh: oriented dan visual / very high level. Contoh Bahasa: Java

Proses pembuatan Program:

  1. Kompilasi

    2. Contoh: C, C++

  2. Interpretasi

    3. Contoh: Pearl, Ruby, Python

  3. Kompilasisekaligus Interpretasi

    4. Contoh: Java

Struktur Dasar Algoritma

  1. Bentuk Secara urutan (Sequential)
  2. Bentuk Pemilihan (Seleksi)
  3. Bentuk Perulangan (Loop)

Operator

Simbol dalam program yang digunakan untuk mengubah suatu nilai.

    Operator dibagi menjadi 3:

  1. Operator Aritmatika

    2. Seperti Penjumlahan, Pengurangan, Perkalian, pembagian bilangan real, bagi sisa, pembagian bilangan bulat.

  2. Operator Logika

    1. And
      2. A B A and B
      True True True
      True False False
      False True False
      False False False
    2. Or
      3. A B A or B
      True True True
      True False True
      False True True
      False False False
    3. NOT
      4. A Not A
      True False
      False True

  3. Operator Relational

    4. Contoh: =, <>, <, <=, >, >=

Nilai Variabel

Besaran dari tipe data yang sudah dikenal

Nilai dapat dimanipulasi dengan cara:

  1. Mengisi nilai kedalam variable

    2. var variable = 5 (Nilai)

  2. Menuliskan nilai ke piranti keluaran

    3. write(variable) (Menuliskan 5 kedalam layar)

  3. Diacu dari suatu nama untuk proses perhitungan / operasi

    4. var jumlah = variable * 5

Nilai dapat berupa:

  1. Konstantan yang diisi langsung

    2. const phi : real = 3,14 (nilai phi)

  2. Isi yang disimpan bariable atau konstanta

    3. var variable = 5 (Nilai)

  3. Hasil perhitungan suatu ekspresi atau hasil yang dikirim oleh fungsi.

    4. var jumlah = variable * 5
